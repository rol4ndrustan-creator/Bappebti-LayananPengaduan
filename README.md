# Bappebti Multi-Role Complaint Ecosystem Demo

A Vite + React demo prototype for Bappebti complaint management across Public Users, Platform/Pialang, Bursa, Kliring, Bappebti Regulator, Association, and Super Admin.

## Run locally

```bash
npm install
npm run dev
```

## Vercel

- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`

Do not upload package-lock.json if it was generated in another environment.
