# Bappebti Multi-Role Complaint Ecosystem Demo

Professional regulator-style demo for Layanan Pengaduan Bappebti.

## Included
- Public landing page with login/register demo flow
- Public dashboard, complaint list, submission wizard, confirmation, success ticket page, notification page, and profile page
- Compact case queues using list/table layout
- Case detail with singular current ownership and vertical timeline
- Separate views for Platform/Pialang, Bursa, Kliring, Bappebti Regulator, Association, and Super Admin
- Dashboard and reporting examples for SLA, critical queue, member performance, kritik/masukan, and reports

## Run
npm install
npm run dev

## Build
npm run build

For Vercel:
Install Command: npm install
Build Command: npm run build
Output Directory: dist
