# Bappebti Complaint Service Demo

This is a static React/Vite prototype for demo purposes.

## Run locally
```bash
npm install
npm run dev
```
Then open the local URL shown in Terminal, usually `http://localhost:5173`.

## Build for production
```bash
npm run build
```
The static files will be generated in the `dist` folder.

## Deploy to Vercel
1. Upload this folder to GitHub.
2. Go to Vercel > Add New Project.
3. Import the GitHub repository.
4. Framework preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

## Demo flow
1. Landing Page
2. Buat Pengaduan
3. Pengaduan Resmi
4. Identitas Pelapor
5. Detail Pengaduan
6. Bukti Pendukung
7. Tinjau & Kirim
8. Konfirmasi Berhasil
9. Cek Status
